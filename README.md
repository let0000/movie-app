# movie-app
TMDB API와 react를 이용하여 만든 현재 상영중인 영화를 볼 수 있는 사이트\

## 기능
상영중인 영화를 리스트로 나타내는 기능\
영화 리스트 클릭 시 해당 영화의 상세 정보 나타내는 기능\
router-dom 과 navigate를 이용한 네비게이션 기능

## 라이브러리
react-router-dom

## 사용 API
TMDB API (학습용으로 더미데이터를 가져와서 구현)
