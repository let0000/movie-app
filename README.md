# movie-app
TMDB API와 react를 이용하여 만든 현재 상영중인 영화를 볼 수 있는 사이트
(학습용으로 실제 데이터를 받아오진 않고 더미데이터를 만들어서 수행)

## 기능
상영중인 영화 조회\
영화 상세 정보 조회\
라우터 돔을 이용한 헤더
